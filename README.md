# LIIKE – Liikunta- ja Hyvinvointikeskus

Modernit nettisivut Liikunta- ja Hyvinvointikeskus Liike Oy:lle (Mänttä).

## Käynnistys

```bash
npm install
npm run dev
```

Avaa selain osoitteessa http://localhost:5173

## Sisältö

Tiedot on koottu osoitteesta [lhk-liike.fi](https://www.lhk-liike.fi/):

- Kuntosali, ryhmäliikunta, sisäpyöräily, valmennus
- Hinnasto (aikakortit, kertamaksut, monikortit)
- Aukioloajat ja yhteystiedot

## Teknologia

- Vite + React + TypeScript
- Tailwind CSS v4
- React Router
